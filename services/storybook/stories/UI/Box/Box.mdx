# Box

Box is the **foundational building block** of our design system. It's a minimal abstraction over HTML elements that provides styling capabilities while maintaining maximum flexibility.

**Think of Box as:** A styling-enabled `<div>` that can become any HTML element.

Box is designed to be:

- **Minimal** - No built-in layout logic or predefined styles
- **Flexible** - Can render as any HTML element via `as` prop
- **Universal** - The base layer for all other components (Stack, Card, Button, etc.)
- **Unopinionated** - You control everything

## Box vs Other Components

| Component  | Purpose                     | Built on          |
| ---------- | --------------------------- | ----------------- |
| **Box**    | General-purpose container   | -                 |
| **Stack**  | Vertical/horizontal layouts | Box + flex logic  |
| **Inline** | Horizontal layouts          | Box + flex(row)   |
| **Center** | Center alignment            | Box + centering   |
| **Card**   | Card-style containers       | Box + card styles |

**All components inherit Box's capabilities**, but add specific behavior or styling.
